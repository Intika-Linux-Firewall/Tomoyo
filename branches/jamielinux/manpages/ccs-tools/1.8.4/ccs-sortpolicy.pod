=head1 NAME

ccs-sortpolicy - domain policy sorting utility for TOMOYO Linux

=head1 SYNOPSIS

B<ccs-sortpolicy>

=head1 DESCRIPTION

This program reads domain policy from standard input and send the sorted contents to standard output. This is similar to B<sort>(1), except that sorting is done per domain.

You can pass the content of F</etc/ccs/domain_policy.conf> or F</proc/ccs/domain_policy> using redirection or pipes to the standard input of this program.

=head1 EXAMPLES

=over 4

=item B<Sort domain policy>

  ccs-sortpolicy < /proc/ccs/domain_policy

=back
