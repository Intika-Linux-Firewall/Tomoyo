=head1 NAME

ccs-setlevel - profile management utility for TOMOYO Linux

=head1 SYNOPSIS

B<ccs-setlevel> I<name=value>

=head1 DESCRIPTION

This program can be used to edit the profiles currently defined in F</proc/ccs/profile>. The output must be quoted or escaped to avoid misinterpretation by the shell.

Before this program can be invoked, you must register it in F</proc/ccs/manager>. After initializing policy, this is usually as simple as rebooting the system.

=head1 EXAMPLES

=over 4

=item B<Change mode for profile number 3 to enforcing mode>

  ccs-setlevel "3-CONFIG={ mode=enforcing }"

=back
