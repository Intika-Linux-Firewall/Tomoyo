=head1 NAME

tomoyo-loadpolicy - load TOMOYO Linux manually

=head1 SYNOPSIS

B<tomoyo-loadpolicy> [I<option>]

B<tomoyo-loadpolicy> [I<option>] [I<remote_ip:remote_port>]

=head1 DESCRIPTION

This program reads TOMOYO Linux policy from standard input and loads it into the kernel.

=head1 OPTIONS

=over 4

=item B<-e>

Append to F</sys/kernel/security/tomoyo/exception_policy>.

=item B<-ef>

Overwrite F</sys/kernel/security/tomoyo/exception_policy>.

=item B<-d>

Append to F</sys/kernel/security/tomoyo/domain_policy>.

=item B<-df>

Overwrite F</sys/kernel/security/tomoyo/domain_policy>.

=item B<-m>

Append to F</sys/kernel/security/tomoyo/manager>.

=item B<-p>

Append to F</sys/kernel/security/tomoyo/profile>.

=item B<-s>

Append to F</sys/kernel/security/tomoyo/stat>.

=item I<remote_ip:remote_port>

Write to policy on a remote system via an agent waiting at port I<remote_port> on IP address I<remote_ip>.

=back

=head1 EXAMPLES

=over 4

=item B<Append a line to exception policy>

  echo "acl_group 0 file read proc:/meminfo" | tomoyo-loadpolicy -e

=item B<Remove a line from exception policy>

  echo "delete acl_group 0 file read proc:/meminfo" | tomoyo-loadpolicy -e

=item B<Append a line to domain policy>

  ( echo "<kernel>"; echo "file execute /sbin/init" ) | tomoyo-loadpolicy -d

=item B<Replace currently loaded domain policy>

  tomoyo-loadpolicy -df < /etc/tomoyo/domain_policy.conf

=item B<Append to policy remotely>

  tomoyo-loadpolicy -d 192.168.1.1:10000 < /etc/tomoyo/192.168.1.1/domain_policy.conf

=item B<Remove a line from C</sys/kernel/security/tomoyo/manager>>

  echo "delete /usr/sbin/tomoyo-queryd" | tomoyo-loadpolicy -m

=back
