=head1 NAME

tomoyo-patternize - utility to convert pathnames in policy into patterns

=head1 SYNOPSIS

B<tomoyo-patternize> I<pattern>

=head1 DESCRIPTION

This program reads domain policy from standard input and replaces pathnames with patterns. The resulting policy is sent to standard output and is useful to shorten and simplify domain or exception policy.

This will not replace pathnames used for domain names themselves, or for execute permissions.

=head1 EXAMPLES

=over 4

=item B<Replace C</tmp/phpfk30dk> with C</tmp/php\?\?\?\?\?\?>>

  tomoyo-patternize '/tmp/php\?\?\?\?\?\?' < /etc/tomoyo/domain_policy.conf > /etc/tomoyo/domain_policy.conf.new

=item B<Replace C</tmp/phpfk30dk> with C</tmp/php\?\?\?\?\?\?>>

  tomoyo-patternize 'file_pattern /tmp/php\?\?\?\?\?\?' < /etc/tomoyo/domain_policy.conf > /etc/tomoyo/domain_policy.conf.new

=item B<Replace C</tmp/phpZjk1k3> with C<@GROUP1>>

  tomoyo-patternize 'path_group GROUP1 /tmp/php\?\?\?\?\?\?' < /etc/tomoyo/domain_policy.conf > /etc/tomoyo/domain_policy.conf.new

=item B<Replace integers between 0-100 or 100-200 with C<@GROUP2>>

  tomoyo-patternize 'number_group GROUP2 0-100' 'number_group GROUP2 100-200' < /etc/tomoyo/domain_policy.conf > /etc/tomoyo/domain_policy.conf.new

=item B<Replace loopback addresses with C<@LOCALHOST>>

  tomoyo-patternize 'address_group LOCALHOST 127.0.0.1' 'address_group LOCALHOST 0:0:0:0:0:0:0:1' < /etc/tomoyo/domain_policy.conf > /etc/tomoyo/domain_policy.conf.new

=back
