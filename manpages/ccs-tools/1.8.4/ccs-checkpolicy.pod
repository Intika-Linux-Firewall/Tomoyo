=head1 NAME

ccs-checkpolicy - policy validator utility for TOMOYO Linux

=head1 SYNOPSIS

B<ccs-checkpolicy> [I<option>]

=head1 DESCRIPTION

This program reads domain policy or exception policy from standard input, and checks for errors. Use this during policy development to ensure that you are implementing correct policy.

=head1 OPTIONS

=over 4

=item B<d>

Interpret input as domain policy.

=item B<e>

Interpret input as exception policy.

=back

=head1 EXAMPLES

=over 4

=item B<Check for errors in domain policy>

  ccs-checkpolicy d < /etc/ccs/domain_policy.conf

=item B<Check for errors in exception policy>

  ccs-checkpolicy e < /etc/ccs/domain_policy.conf

=back
